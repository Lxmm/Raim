关于

（1）机场NPA预测：依据NPA的标准（完好性报警水平门限为556m(0.3海里)）对全国185个机场进行3天的RAIM预测，将RAIM预测结果通过图形和文本两种方式显示出来。用户选择“机场”（最多至5个）、“仰角”和输出形式后点击“预测”即可输出机场NPA的预测结果。


（2）终端区预测：依据完好性报警水平门限1海里，对全国185个机场进行3天的RAIM预测，将RAIM预测结果通过图形和文本两种方式显示出来。用户选择“机场”（最多至5个）、“仰角”和输出形式后点击“预测”即可输出终端区的预测结果。


（3）GPS卫星状态信息：显示预测时间范围内（从当前时间开始的3天）GPS卫星信息，包括卫星历书给出的信息和来自NANU和NOTAM报文的卫星不可用信息。


（4）区域预测：给出指定区域按网格划分后对特定网格的预测结果，预测时根据指定的精度标准进行，目前给出了中国区域的网格预测结果。该种预测包括：


n  静态预测：给定时间点的区域网格预测。用户选择预测时间点后点击“开始预测”即可输出指定时间点的中国区域的预测结果。


n  统计预测：给定时间段的区域网格统计预测，采用最长不可用时间作为统计量，既给定的时间段内某区域网格最长的不可用时间。用户选择预测起始时间点和结束时间点后点击“开始预测”即可输出指定时间区段内中国区域各网格内最长的不可用时间。


（5）航路预测功能：针对给出配置的5条航路进行RAIM预测，给出相应的预测结果。预测时将航路划分为航段，针对每个航段进行预测。


n  航路预测：根据用户指定的预测标准，给出指定航路在24小时内的可用性信息，将24小时分时段的基础预测数据与标准进行比对，如果低于标准则该时段的航路不可用。


n  航路分析：根据用户指定的航路和预测时段将其基础预测结果按飞行时段进行区段结果的显示。
